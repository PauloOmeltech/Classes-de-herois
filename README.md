## 🛡️ Hero - Adventure Class in Python

A simple generic class to represent heroes in adventure games or RPGs.

## 📋 About the Project

This project contains the `Hero` class, which allows you to create characters with different types (mage, warrior, monk, and ninja), 
each with their own unique attack style.

Perfect for beginners learning Object-Oriented Programming (OOP) or as a foundation for simple game systems.

## ✨ Features

- Create heroes with name, age, and type
- `attack()` method with different behavior depending on the hero's type
- Clean, well-commented, and easy-to-extend code
- Support for adding new hero types

## 🗡️ Available Hero Types

| Type       | Attack              |
|------------|---------------------|
| Mage       | magic               |
| Warrior    | sword               |
| Monk       | martial arts        |
| Ninja      | shuriken            |

# 🛡️ Herói - Classe de Aventura em Python

Uma classe genérica simples para representar heróis de jogos de aventura ou RPG.

## 📋 Sobre o Projeto

Este projeto contém a classe `Heroi`, que permite criar personagens com diferentes tipos (mago, guerreiro, monge e ninja), 
cada um com seu próprio estilo de ataque.

Ideal para iniciantes em Programação Orientada a Objetos (POO) ou como base para sistemas de jogos simples.

## ✨ Funcionalidades

- Criação de heróis com nome, idade e tipo
- Método `atacar()` com comportamento diferente conforme o tipo do herói
- Código limpo, comentado e fácil de expandir
- Suporte a novos tipos de heróis

## 🗡️ Tipos de Heróis Disponíveis

| Tipo       | Ataque              |
|------------|---------------------|
| Mago       | magia               |
| Guerreiro  | espada              |
| Monge      | artes marciais      |
| Ninja      | shuriken            |
